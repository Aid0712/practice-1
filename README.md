# Python Junior Roadmap (3–4 месяца)

Этот репозиторий — учебный план и мини‑проекты для начинающих.
Всё написано простыми словами на русском языке, с короткими объяснениями.

## Что внутри
- Полный план обучения по шагам.
- Папки с темами, примерами кода и короткими комментариями.
- Мини‑проекты, включая темы, близкие к современному ИИ.
- Глоссарий всех терминов, которые используются в проектах.
- Простое объяснение Git/GitHub/GitLab.
- База по backend с наглядными схемами.

## Структура
- `learning-plan/plan.md` — подробный план на 3–4 месяца.
- `learning-plan/glossary.md` — глоссарий простыми словами.
- `learning-plan/git-tools.md` — кратко про Git, GitHub и GitLab.
- `learning-plan/backend-notes.md` — основы backend простыми словами.
- Папки `01-*` ... `11-*` — темы и примеры.

## Как работать
1. Открывайте `learning-plan/plan.md` и идите по шагам.
2. В каждой папке есть `README.md` и Python‑файлы с примерами.
3. Мини‑проекты в `11-mini-projects` — применяйте знания на практике.

Удачи! Всё получится.
