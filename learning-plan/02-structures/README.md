# 02 — Коллекции данных

## Простое объяснение
Коллекции помогают хранить много значений сразу.

- **Список** — упорядоченный набор.
- **Словарь** — пары «ключ → значение».
- **Множество** — только уникальные значения.

## Визуально
```
Список:   ["учиться", "сон", "спорт"]
Словарь:  {"name": "Аня", "age": 18}
Множество:{1, 2, 3}
```

## Что внутри папки
- `collections.py` — примеры работы с коллекциями.
