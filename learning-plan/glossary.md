# Глоссарий (простыми словами)

- **Python** — язык программирования, на котором мы пишем код.
- **Интерпретатор** — программа, которая читает и выполняет код Python.
- **Переменная** — «коробка» для хранения значения.
- **Тип данных** — вид значения (число, строка и т.д.).
- **Строка** — текст в кавычках.
- **Булево значение** — правда/ложь (`True`/`False`).
- **Функция** — маленькая часть программы с понятной задачей.
- **Аргумент** — входные данные функции.
- **Возврат** — то, что функция отдаёт назад.
- **Класс** — шаблон для создания объектов.
- **Объект** — конкретный экземпляр класса.
- **ООП** — стиль программирования на основе классов и объектов.
- **SOLID** — набор правил, чтобы код был понятным и гибким.
- **KISS** — «делай проще», не усложняй.
- **Чистый код** — код, который легко читать и менять.
- **Модуль** — файл с кодом.
- **Пакет** — папка с модулями.
- **Тест** — автоматическая проверка, что код работает.
- **API** — способ общаться между программами.
- **HTTP** — правила обмена данными в интернете.
- **JSON** — простой формат данных (похоже на словарь).
- **FastAPI** — фреймворк для создания API на Python.
- **Django** — фреймворк для сайтов на Python.
- **База данных** — место для хранения данных.
- **SQL** — язык запросов к базе данных.
- **CRUD** — создать, прочитать, обновить, удалить.
- **Логирование** — запись событий в файл или консоль.
- **Асинхронность** — умение делать несколько задач «почти одновременно».
- **Вектор** — список чисел (часто используется в ИИ).
- **Схожесть** — насколько тексты похожи друг на друга.
- **TF-IDF** — простой способ выделять важные слова.
- **Git** — инструмент для хранения истории изменений.
- **GitHub** — сайт для хранения Git-проектов.
- **GitLab** — сервис, похожий на GitHub.
