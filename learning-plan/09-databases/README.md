# 09 — Базы данных

## Простое объяснение
База данных — это как тетрадь, где хранятся записи.
SQLite — встроенная база, удобна для обучения.

## Визуально
```
Таблица notes
+----+-----------+
| id | text      |
+----+-----------+
|  1 | Привет    |
+----+-----------+
```
