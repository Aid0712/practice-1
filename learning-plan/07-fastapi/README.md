# 07 — FastAPI (мини‑API)

## Что такое API
API — это «меню» сервера. Ты выбираешь блюдо (эндпоинт) и получаешь результат.

## Мини‑задача
Сделать API для заметок (пока всё хранится в памяти).

## Визуально
```
POST /notes  -> создать заметку
GET  /notes  -> получить список
```

## Что внутри папки
- `app.py` — простой сервер на FastAPI.
