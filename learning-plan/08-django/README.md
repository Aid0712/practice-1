# 08 — Django (мини‑проект)

## Простое объяснение
Django помогает быстро делать сайты.
У него есть готовые инструменты для базы и страниц.

## Что такое MVT
В Django используется **MVT**:
- **Model** — данные (таблицы).
- **View** — логика.
- **Template** — HTML‑шаблон.

## Что внутри папки
- `prompt_library/models.py` — модель промпта.
- `prompt_library/views.py` — простой ответ.
- `prompt_library/urls.py` — путь на страницу.
