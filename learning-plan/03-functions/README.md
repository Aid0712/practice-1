# 03 — Функции

## Зачем нужны функции
Функции помогают делить большую задачу на маленькие.
Так код проще читать и исправлять.

## Мини‑пример
```python
def add(a, b):
    return a + b
```

## Важно помнить
- Одна функция — одна задача (KISS).
- Названия должны быть понятны.

## Что внутри папки
- `functions.py` — простые функции.
