# 04 — ООП и SOLID

## Простое объяснение ООП
- **Класс** — шаблон.
- **Объект** — конкретный экземпляр шаблона.

Пример:
- Класс `User`.
- Объект: `user = User("Аня")`.

## SOLID — как запомнить
- **S** — один класс = одна ответственность.
- **O** — расширяем без поломки старого.
- **L** — наследник ведёт себя как родитель.
- **I** — не заставляй делать лишнее.
- **D** — зависим от абстракций.

## Мини‑пример идеи S
```python
class User:
    pass

class EmailSender:
    pass
```
