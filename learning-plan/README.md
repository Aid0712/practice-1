# Учебный план и материалы

Этот раздел содержит весь учебный план, примеры кода и мини‑проекты.

## Файлы
- `plan.md` — подробный план по неделям.
- `glossary.md` — глоссарий терминов простыми словами.
- `git-tools.md` — что такое Git, GitHub и GitLab.
- `backend-notes.md` — основы backend с простыми схемами.

## Папки и что внутри

### `01-basics`
**Тема:** переменные, типы, ввод/вывод.
**Файлы:**
- `README.md` — объяснения и мини‑шпаргалка.
- `main.py` — самый простой пример.

### `02-structures`
**Тема:** списки, словари, множества.
**Файлы:**
- `README.md` — что такое коллекции.
- `collections.py` — базовые операции.

### `03-functions`
**Тема:** функции и разбиение задач.
**Файлы:**
- `README.md` — зачем нужны функции.
- `functions.py` — простые функции.

### `04-oop-solid`
**Тема:** классы, объекты, SOLID.
**Файлы:**
- `README.md` — объяснение SOLID.
- `oop.py` — мини‑пример с разделением ответственности.

### `05-testing`
**Тема:** тесты.
**Файлы:**
- `README.md` — зачем нужны тесты.
- `calc.py` — функция для теста.
- `test_calc.py` — тест к функции.

### `06-async`
**Тема:** асинхронность.
**Файлы:**
- `README.md` — что такое async.
- `async_demo.py` — очень простой пример.

### `07-fastapi`
**Тема:** API на FastAPI.
**Файлы:**
- `README.md` — мини‑описание API.
- `app.py` — простой API для заметок.

### `08-django`
**Тема:** Django проект (каркас).
**Файлы:**
- `README.md` — объяснение Django.

### `09-databases`
**Тема:** базы данных и SQLite.
**Файлы:**
- `README.md` — очень простые идеи.

### `10-api`
**Тема:** работа с API и JSON.
**Файлы:**
- `README.md` — что такое запрос/ответ.

### `11-mini-projects`
**Тема:** мини‑проекты (похожие на задачи ИИ).
