# 01 — Базовые вещи

## Что это такое
Здесь самые первые шаги: переменные, типы и ввод/вывод.
Я объясняю как студенту первого курса.

## Простое объяснение
- **Переменная** — коробка для значения.
- **Тип данных** — что внутри коробки: число или текст.
- **print()** — показать на экране.
- **input()** — спросить у пользователя.

## Мини‑пример (самый простой)
```python
name = "Аня"
print("Привет,", name)
```

## Что внутри папки
- `main.py` — первый пример с вводом и выводом.
