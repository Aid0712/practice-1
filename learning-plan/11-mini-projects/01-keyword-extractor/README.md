# Проект 1 — Выделение ключевых слов

## Идея простыми словами
Мы берём текст и ищем самые частые важные слова.
Это похоже на первый шаг подготовки данных для ИИ.

## Шаги
1. Убрать знаки и сделать текст нижним регистром.
2. Убрать короткие слова.
3. Посчитать, что встречается чаще всего.

## Мини‑пример
```
Текст: "Python учит писать код"
Результат: ["python", "писать", "код"]
```

Смотри `keyword_extractor.py`.
