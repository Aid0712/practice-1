# Проект 3 — FastAPI сервис анализа текста

## Идея простыми словами
Мы делаем API, которое принимает текст и возвращает результат анализа.
Например: длина текста и простая оценка «настроения».

## Шаги
1. Создать эндпоинт `/analyze`.
2. Посчитать длину текста.
3. Посчитать простую оценку по словам.

## Визуально
```
POST /analyze
{ "text": "мне хорошо" }
-> { "length": 10, "score": 1 }
```

Смотри `app.py`.
