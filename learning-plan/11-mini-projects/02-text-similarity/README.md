# Проект 2 — Поиск похожих текстов

## Идея простыми словами
Хотим понять, насколько два текста похожи.
Если у них много одинаковых слов — они близки.

## Шаги
1. Превращаем текст в набор слов.
2. Считаем «похожесть» по формуле.

## Визуально
```
Текст A: python код быстро
Текст B: python код
=> похожесть высокая
```

Смотри `similarity.py`.
