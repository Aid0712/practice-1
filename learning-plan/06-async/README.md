# 06 — Асинхронность

## Простое объяснение
Асинхронность — это когда программа не «зависает», пока ждёт.
Например, ждёт ответ от сервера.

## Визуально
```
Обычный код: ждём -> потом делаем
Async код: ждём, но можем делать другое
```

## Что внутри папки
- `async_demo.py` — короткий пример с `await`.
